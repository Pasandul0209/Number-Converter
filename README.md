\# Number Converter Project (Java)



This is a simple Java console-based menu program that allows users to select different number system converters such as Decimal to Binary, Octal, and Hexadecimal.



Features

\- Menu-driven system

\- Loop-based navigation

\- Options to go back (0) and exit (99)

\- User input using Scanner



Technologies Used

\- Java

\- Scanner class

\- Loops (while, if-else)
